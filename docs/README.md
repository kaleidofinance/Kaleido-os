## 🔒 Security

All smart contracts have been audited for:
- Reentrancy protection
- Flash loan attacks
- Access control
- Decimal precision
- Fee manipulation
- Collateral management
- Dust attacks
- Front-running

- **Code blocks**: Used for commands, code snippets, and examples
